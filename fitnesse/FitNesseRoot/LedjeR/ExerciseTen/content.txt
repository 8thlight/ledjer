Ledger, Part Ten
Ledger.save() serializes to a file
Ledger.load() deserializes

We create a ledger, make some transactions, and save.  Then we add one more transaction without saving.
 
|script|empty ledger|
|deposit|1000|cents on|Jan 4, 2013|
|pay|500|cents to|Apple|on|Feb 5, 2013|
|save|
|deposit|1234|cents on|Feb 6, 2013|

Now load the ledger and make sure it only has the first two transactions.

|script|
|load|

|Query:Statement|
|lineitem|
|Deposit: $10.00|
|Payment to Apple: ($5.00)|
|Total: $5.00|
